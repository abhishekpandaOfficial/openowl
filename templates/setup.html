Setup Guide

Fast local setup for desktop-first development and Telegram webhook testing.

🛠️ 1) Install and configure

Copy environment, fill tokens, and ensure Ollama/Groq keys are available.

cp .env.example .env # edit .env for TELEGRAM_BOT_TOKEN, GROQ_API_KEY, OWNER_ID

🚀 2) Start dependencies

docker-compose up -d postgres redis source .venv/bin/activate uvicorn main:app --reload --port 8000

🌐 3) Expose webhook + verify

ngrok http 8000 # set TELEGRAM_WEBHOOK_URL in .env # restart app curl http://localhost:8000/health
Open Desktop UI Check API Docs
